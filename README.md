# To-do List App

A modern and responsive To-do List web application built using **HTML5, CSS3, and JavaScript**.  
This project allows users to efficiently manage daily tasks through a clean and interactive user interface.

---

## 📌 Features

- Add new tasks instantly
- Mark tasks as completed
- Delete tasks easily
- Clean and responsive user interface
- Keyboard support (Press **Enter** to add tasks)
- Smooth and user-friendly task management experience

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and responsive layout  
- **JavaScript (Vanilla JS)** – Functionality and DOM manipulation
